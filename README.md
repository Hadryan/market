Ortolang Market
=============

Install
-------

    npm install -g grunt
    npm install -g grunt-cli
    npm install -g yo

    npm install
    bower install

Preview the Ortolang Market app (with Livereload).
-------

    grunt serve

Run the unit tests
-------

    grunt test

Build an optimized, production-ready version of the Ortolang Market app.
-------

    grunt
